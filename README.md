# Catxls package
catxlsx: Reading xlsx on server same cat command
# Installation

```bash
pip install catxls
```
# How to use

```bash
catxls -h
catxls <file xls or file xlsx>
```
# Development guide
## 1. requirement
- virtualenv
- python 3+

## 2. setup
- clone project and run `python setup.py develop`
- test by run it `catxls -h`
